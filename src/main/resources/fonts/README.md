# Fuentes Unicode para PDFBox

Descarga las fuentes Noto Sans desde Google Fonts y colócalas en esta carpeta:

1. Visita: https://fonts.google.com/noto/specimen/Noto+Sans
2. Descarga la familia completa
3. Extrae y copia estos archivos aquí:
   - `NotoSans-Regular.ttf`
   - `NotoSans-Bold.ttf`
   - `NotoSans-Italic.ttf`
   - `NotoSans-BoldItalic.ttf`

Alternativamente, usa wget:
```bash
cd src/main/resources/fonts/
wget https://github.com/googlefonts/noto-fonts/raw/main/hinted/ttf/NotoSans/NotoSans-Regular.ttf
wget https://github.com/googlefonts/noto-fonts/raw/main/hinted/ttf/NotoSans/NotoSans-Bold.ttf
wget https://github.com/googlefonts/noto-fonts/raw/main/hinted/ttf/NotoSans/NotoSans-Italic.ttf
wget https://github.com/googlefonts/noto-fonts/raw/main/hinted/ttf/NotoSans/NotoSans-BoldItalic.ttf
```
